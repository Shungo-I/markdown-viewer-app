# タイポグラフィ

デジタル庁デザインシステムに基づいたタイポグラフィーシステムです。

## フォントファミリー

### Base Font
- **Noto Sans JP**: 日本語と英語の基本フォント
- **フォールバック**: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Helvetica Neue', Arial, sans-serif

### Monospace Font
- **Noto Sans Mono**: コード表示用のモノスペースフォント
- **フォールバック**: 'SFMono-Regular', Consolas, 'Liberation Mono', Menlo, Courier, monospace

## 使用方法

CSSカスタムプロパティとして定義されているため、以下のように使用できます：

```css
/* 見出しスタイルの適用 */
.heading-large {
  font: var(--font-heading-xl);
  color: var(--color-text-primary);
}

/* 本文スタイルの適用 */
.body-text {
  font: var(--font-body-md);
  color: var(--color-text-primary);
}

/* コードスタイルの適用 */
.code-block {
  font: var(--font-code-md);
  background-color: var(--color-background-secondary);
  padding: 16px;
  border-radius: 8px;
}
```

## 特徴

- **一貫性**: すべてのフォントサイズと行間が体系的に定義されています
- **レスポンシブ**: 異なるデバイスで適切に表示されます
- **アクセシビリティ**: 十分なコントラストと読みやすさを確保
- **拡張性**: 新しいスタイルを追加しやすい設計
- **デジタル庁準拠**: デジタル庁デザインシステムの原則に従っています
